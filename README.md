# Kvadratik Pro 4.0

Пиксельный редактор с AI помощником на базе DeepSeek.

## 📥 СКАЧАТЬ ПРОГРАММУ

[⬇️ Скачать с Яндекс.Диска](https://disk.yandex.ru/d/Xu4DqZOEypRyHQ) (30 МБ)

Программа не требует установки. Просто запусти `kvadratik.exe`

## 🎨 ВОЗМОЖНОСТИ

- 🖌️ 5 видов кистей (круг, текстура, спрей, звезда)
- 🎭 Анимация и экспорт в GIF
- 📦 Слои с прозрачностью
- 📝 Текст (пиксельный шрифт)
- 🤖 AI помощник (DeepSeek-R1:7b)
- ✨ Фильтры и эффекты
- 🎨 Color Picker и история цветов

## ⚙️ УСТАНОВКА AI

Для работы AI помощника нужно установить Ollama:

1. Скачай [Ollama](https://ollama.com)
2. Установи модель:
   ```bash
   ollama pull deepseek-r1:7b     # 4.7 ГБ, умный
   ollama pull deepseek-r1:1.5b   # 1.1 ГБ, легкий
   ollama pull gemma3:4b          # 3.3 ГБ, от Google
   ollama pull llama3.2:3b        # 2.8 ГБ, от Meta
