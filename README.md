#  CeTzY Dev — Мои программы для Windows

Привет! Я CeTzY, разработчик полезных программ для Windows. Все мои программы бесплатны и не требуют установки.

##  Мои проекты

### [1C Cache Cleaner](1c-cleaner/)
Чистильщик кэша 1С. Автоматически находит и удаляет временные файлы 1С, ускоряя работу программы.

**Возможности:**
-  Автоматический поиск кэша
-  Расчет размера папок
-  Выборочная очистка
-  Статистика очисток

### [Kvadratik Pro](kvadratik-pro/)
Пиксельный редактор с анимацией и слоями. Идеально для pixel art.

**Возможности:**
-  5 видов кистей
-  Анимация с кадрами
-  Фильтры и эффекты
-  Горячие клавиши
